Fizz Buzz

Fizz Buzz is a simple algorithm that evaluates all the numbers between 1 and a given limit for any numbers divisible by 3 and/or 5.
Matches are indicated as follows:
  • "Fizz" is output if a number is only divisible by 3
  • "Buzz is output if a number is only divisible by 5
  • "FizzBuzz is output if a number is divisible by both 3 and 5
Example: Given an upper limit of 15, the following would be output:
1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, Fizz Buzz
